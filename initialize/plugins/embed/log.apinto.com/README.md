## 基本介绍
日志检索提供基本的针对apinto节点的日志检索能力
## 功能特性
- 针对节点，查询所有文件类型的日志列表
- 支持追踪节点日志、请求Access日志
- 支持在控制台下载日志，无需进入服务器操作
- 按日志对象监视新增日志
